Introduction to HTML
.
HTML basically stands for Hyper text markup language.
This is the language for the creation of a basic webpage.
We can use CSS, PHP, JAVASCRIPT or BOOTSTRAP for enhancing
the website visually.
.
Till now, we have studied about some of the basic tags,
that we can use in our code for assigning and defining the elements.
Some of them are as follows:-
.
< h1 > to < h6 > are used to define the font size in increasing order from h1 to h6.
< iframe > is used to inculcate a file or a webpage in the website.
< link > is used to add an address link in a particular image or text.
< img src > is used to add an image to the webpage.
< marquee > tag is used to give an animated look to the element attached to the tag.
.
Padding has a different meaning than border.
Border is the outermost layer that gives the block, an orientation
whereas
Padding is the area between the text and the border.
.
The basic syntax of an HTML code is as follows:-
< !DOCTYPE HTML >
< HTML >
< head >
< /head >
< body >
< /body >
< /HTML >
.
The first statement is to define the HTML version used in the code.
Head contains the meta data which is used to find a page, and to define css styles and javascript scripts
Body consists of all the data, shown on the page.
